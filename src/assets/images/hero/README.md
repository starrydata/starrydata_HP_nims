# Hero / banner images

トップページ・各ページの hero 用大判画像。

## 推奨仕様

| 項目 | 推奨 |
|---|---|
| サイズ | 1920 × 1080 px (16:9) もしくは 2400 × 1200 px (2:1) |
| 形式 | JPG |
| 容量 | 500 KB 以下 |
| 圧縮 | mozjpeg quality 75-80 |

## 候補ファイル

- `home.jpg` — トップページ Hero（現状はグラデのみ）
- `collaborate.jpg` — `/collaborate` ページ hero
- `about.jpg` — `/about` ページ hero
