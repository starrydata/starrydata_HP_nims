# Partner / Funder logos

支援機関・パートナー・スポンサー・所属機関のロゴ。

## 命名規則

`<org_slug>.svg`（推奨）／ `<org_slug>.png`（透過）

例:
- `nims.svg`
- `riken_aip.svg`
- `jst.svg`
- `university_of_tsukuba.svg`
- `sakura_internet.svg`

## 推奨仕様

| 項目 | 推奨 |
|---|---|
| 形式 | **SVG**（ベクター・推奨）or PNG（透過、最低 200px 高さ） |
| 高さ | 表示時に 36-48px。元画像は 200px 以上 |
| 色 | 機関の公式版を使用 |
| ライセンス | 各機関のロゴ使用ガイドラインを確認 |

- `/collaborate` ページ
