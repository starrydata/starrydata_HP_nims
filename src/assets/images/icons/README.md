# Icons

サイト内のアイコン素材（favicon、UI アイコンなど）。

## ファイル

- `favicon.ico` — ブラウザタブ用
- `favicon-32.png` — 32×32
- `favicon-180.png` — Apple Touch Icon 180×180
- `logo.svg` — サイトロゴ（SVG）
- `logo-mark.svg` — ロゴマーク（星マーク）のみ

## 推奨

- SVG をベースに、PNG は必要なサイズに書き出す
- favicon は [realfavicongenerator.net](https://realfavicongenerator.net/) で生成
